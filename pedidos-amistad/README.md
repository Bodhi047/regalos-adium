# 🎉 Pedidos Especiales — Día de la Amistad

Una pequeña broma web para revelar el pedido de regalo del Día de la Amistad. 😄

La página muestra un "cargando pedidos" con un botón. Hay que hacer **100 clicks** (¡sí, cien!) antes de que revele —al azar— una persona y el regalo que pidió. Con confeti, emojis y sonidito incluido.

## 🌐 Ver online

Una vez publicado con GitHub Pages, quedará disponible en:

```
https://<tu-usuario>.github.io/pedidos-amistad/
```

## 🖥️ Probar localmente

Simplemente abrí `index.html` en cualquier navegador (doble click).
