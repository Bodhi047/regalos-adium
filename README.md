# 🎁 Regalos Adium - Día de la Amistad

Página web interactiva para el sorteo de **"Pedidos Especiales - Día de la Amistad"** en Adium.

Una divertida broma donde necesitás **100 clicks** para que se carguen los pedidos de tus compañeros 😂

## 🎯 ¿Qué es?

Una página HTML standalone que simula un "bug en el sistema" donde:
- El usuario ve un spinner girando
- Recibe frases motivacionales con cada click
- Necesita hacer exactamente **100 clicks** para que se cargue la lista de pedidos
- Cuando finalmente carga, muestra:
  - **Nombre de la persona** que pidió
  - **Qué pidió** (regalo deseado)
  - Efectos de confetti y celebración 🎉

## 🚀 Cómo usar

### Opción 1: Abrir localmente
1. Descargá `bug_pedidos_final.html`
2. Haz doble click en el archivo
3. Se abre en tu navegador predeterminado
4. ¡A bromear!

### Opción 2: Usar en línea
Si lo subes a GitHub Pages o cualquier servidor web:
```
https://tu-github.com/regalos-adium/bug_pedidos_final.html
```

### Opción 3: Compartir el link directo
Podés compartir el archivo raw desde GitHub:
```
https://raw.githubusercontent.com/TU_USUARIO/regalos-adium/main/bug_pedidos_final.html
```

## 🎮 Controles

- **Click en botón:** Avanza 1 click
- **Presiona ENTER o SPACE:** También cuenta como click
- **Botón "Cargar otro pedido":** Reinicia desde 0 clicks

## 🛠️ Personalización

Si querés modificar la cantidad de clicks requeridos, busca en el código:
```javascript
const clicksRequeridos = 100;
```

Y cambia `100` por el número que quieras.

## 🎉 La Broma

La idea es compartir el link con tus compañeros y que intenten cargar los pedidos. Nadie sabrá cuántos clicks necesita, solo verán:
- Un spinner girando
- Mensajes motivacionales cambiantes
- Un botón que parece no funcionar

Después de 100 clicks... ¡SORPRESA! Se carga el verdadero pedido de alguien al azar. 😂

## 📝 Notas

- Cada vez que reinician, el pedido que carga es diferente (aleatorio)
- No hay progreso visible, solo la magia del 100
- Perfecta para pasar un rato divertido en el trabajo
